## Backup files
Backup file location and formatting are quite important. 

### Location
Backup files are located in the _Backups_ folder in the Desktop of the VersaView. Unless you are contributing to the app's development, we suggest not to modify the name of this folder or the location of the backup files.

### Formatting
The name of a backup file <u>must</u> have the following format:

_STX\<two-digit-switch-number\>backup_

<u>Example:</u> _STX02backup_

Any backup files that do not follow this formatting <u>will be ignored by the app</u>. If at any time you create a backup file and the switch does not appear in the app, this might be the reason.
