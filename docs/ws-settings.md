## Workspace settings
The following environment variables have already been set in the VersaView of the Communications Wall:
- _STX_USER:_ The username for the Stratix switches
- _STX_PWD:_ The corresponding password

These variables can be created by using the following commands in the Windows terminal:
```
setx STX_USER "enter-username-here"
```
```
setx STX_PWD "enter-password-here"
```
